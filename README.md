# RU:
## Введение
Данный репозиторий посвящён курсовой работе на тему **«Механизмы преобразования естественной речи в заданные шаблоны»**.

Цель проекта — разработать систему, которая принимает аудиофайл с диктовкой математических выражений, формул и символов, а затем преобразует их в удобочитаемый вид — таким образом, как человек записал бы их в тетради под диктовку.

---

## Текущее состояние проекта
На данный момент для распознавания речи выбрана модель **Whisper Medium**.  
Подробнее о модели можно прочитать по ссылке:  
https://huggingface.co/openai/whisper-medium

Whisper Medium была выбрана потому, что:
- она достаточно хорошо справляется с распознаванием диктовок математического содержания,
- при этом менее ресурсоёмкая по сравнению с более крупными моделями семейства Whisper.

---

## Нерешённые проблемы
1. Модели Whisper имеют ограничение по длине входного аудио — **до 30 секунд**.  
2. На данный момент **ещё не выбрана модель**, которая будет преобразовывать распознанный текст в привычную математическую запись (дроби, интегралы, суммы и т.п.).

---

## Структура проекта
- **whisper.py** — файл с кодом, который выполняет распознавание речи с помощью Whisper Medium. При желании модель можно заменить на любую другую из семейства Whisper. Внутри файла присутствуют комментарии, объясняющие ключевые части кода.
- **Zadachi.wav** — пример аудиофайла, используемого для тестирования работы системы.
- **result.txt** — текст, полученный от модели после распознавания аудио.
